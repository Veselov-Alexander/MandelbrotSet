# Mandelbrot Set
C++ implementation with OpenGL

## Controls (English keyboard layout is important)
* **W, A, S, D** - camera move
* **E / Q** - zoom in / out
* **X** - export image
* **- / +** - decrease / increase the maximum number of iterations
* **[ / ]** - decrease / increase image export multiplier
* **1, 2, 3, 4** - select display mode
